Crochet Deer Blanket Progress

This website tracks my Corner-to-Corner crochet cabin blanket from creating the pattern to setting up the yarn and crocheting the final design.

My GitHub Repository

1. Pattern Design

The first week consisted of creating the pattern while learning how to work diagonally as opposed tothe traditional horizontal technique.

2. Yarn Setup

I settled on the brand of yarn that came in the biggest skein and set them on my desk so I would be motivated to start the blanket.

3. Starting the Crochet

The project began with the first triangular gauge swatch, which was later discovered to be wrong.

4. Early Progress

After several rows, the plaid pattern started to appear. This is where the initial excitement turned to dread as I realized my gauge was way off

5. Getting Comfortable

As the blanket progressed I had to add more colors but also keep the yarn organized; this is ultimately where I was introduced to yarn bobbins.

6. The Messy Middle

After zooming along and trying to cut corners during color changes, I ended up needing to frog 3 rows to fix the tiniest mistake. This led to manmade yarn vomit.

7. Deer Silhouette Begins

The silhouette was starting to come along but with the addition of more bobbins and colors I started to get overwhelmed by the tails and keeping the bobbins organized. Thankfully, Dollar Tree came through and had a dowel and clothes pins which helped upgrade my set up tremendously.

8. Large Progress

The deer shape became more visible as the project grew. This is where I started to feel like I knew what I was doing and started to finally cruise along.

9. Current Progress

The blanket now covers most of my workspace which makes it a bit harder to work with and keep track of what I'm doing. This led to a major mental meltdown and by the time I came back to real life, I had frogged 12 rows. WHY? The blanket started to unravel because I forgot to secure two tails together.
